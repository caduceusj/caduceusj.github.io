The Supervisor


This game was made during the Game Jam IP Challenge 2023 by me and the Bizarre Programming team, the game is already released at Itch io.
In this game, the protagonist is hired by a company to test some games that are apparently suspected of stealing assets belonging to it. You must play the games and check whether there are stolen assets present or not, and notify the company accurately about which assets were stolen, ensuring intellectual property rights and combating the illegal appropriation of these assets for commercialization, safeguarding the company's rights over its products.
It is an electronic simulation game where the player uses a platform similar to a home computer desktop to access the game software and the assets folder, searching and selecting the stolen ones, while also playing the mini-games.
It was developed in the Unity 2D engine, and the sound design was done using the FMOD Studio and Reaper programs.
