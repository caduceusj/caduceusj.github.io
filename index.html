Current Game Project

Project 157


Project 157 is a horror game with a 3D Low Poly and Low Resolution PS1 style Graphics developed by me and my team Caranguedev using the Godot 4.0 game engine, in this project i am responsible for the programming and level design The game is set in a Brazillian upstate inspired city, the protagonist is a common worker going to his house walking after his Bus breaks in the midway home, little does he know that 2 criminals are riding around the city mugging and killing people, the development of the game is still in Early Stages there is a test gameplay down below showing some of the development progress made, feel free to check it out.



Baroneza
